# Snorre's ZSH config

Various aliases, functions, and settings for ZSH.

## Installation

### [Antidote](https://getantidote.github.io/)

```shell
vim ${ZDOTDIR:-~}/.zsh_plugins.txt
```

```sh
# To install aliases
snorremd/zsh path:aliases


# To install Rustic plugin
snorremd/zsh path:plugins/rustic
```
